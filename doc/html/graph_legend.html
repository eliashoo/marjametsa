Kaavion selite

Tämä sivu selittää, kuinka doxygenin luomia kaavioita tulkitaan.

Ajattele seuraavaa esimerkkiä:

/*! Näkymätön luokka katkaisun vuoksi */
class Nakymaton { };
/*! Katkaistu luokka, periytymissuhde on piilotettu */
class Katkaistu : public Nakymaton { };
/* Luokkaa ei ole dokumentoitu doxygen-kommenteilla */
class Dokumentoimaton { };
/*! Julkista periyttämistä käyttäen periytetty luokka */
class JulkinenKanta : public Katkaistu { };
/*! Malliluokka */
template<class T> class Malli { };
/*! Suojeltua periytymistä käyttäen periytetty luokka */
class SuojeltuKanta { };
/*! Yksityistä periytymistä käyttäen periytetty luokka */
class YksityisKanta { };
/*! Luokka jota periytetty luokka käyttää */
class Kaytetty { };
/*! Kantaluokka joka periytyy useasta muusta luokasta */
class Periytetty : public JulkinenKanta,
: protected SuojeltuKanta,
: private YksityisKanta,
: public Dokumentoimaton,
: public Malli<int>
{
private:
Kaytetty *m_kaytettyLuokka;
}

Tuloksena on seuraavanlainen kaavio:

Ylläolevassa kaaviossa laatikoilla on seuraavat merkitykset :

  • Täytetty harmaa laatikko esittää tietuetta tai luokkaa, jolle kaavio on luotu.
  • Mustareunainen laatikko merkitsee dokumentoitua tietuetta tai luokkaa.
  • Harmaareunainen laatikko merkitsee dokumentoimatonta tietuetta tai luokkaa.
  • Punareunainen laatikko merkistee dokumentoitua luokkaa tai structia jolle ei näytetä kaikkia periytymis-/sisällyssuhteita. Kaavio katkaistaan, jos se ei mahdu määriteltyjen rajojen sisään.

Nuolilla on seuraavat merkitykset:

  • Tummansinistä nuolta käytetään osoittamaan julkista periytymissuhdetta kahden luokan välillä.
  • Tummanvihreää nuolta käytetään suojellussa periytymisessä.
  • Tummanpunaista nuolta käytetään yksityisessä periytymisessä.
  • Purppuranväristä katkoviivaa käytetään, jos luokka sisältyy tai on toisen luokan käyttämä. Nuoli nimetään sillä muuttujalla/muuttujilla jonka läpi osoitettua luokkaa tai tietuetta käytetään.
  • Keltainen katkoviivalla piirretty nuoli merkitsee suhdetta mallin esiintymän ja malliluokan välillä. Nuoli nimetään mallin esiintymän malliparametrilla.
